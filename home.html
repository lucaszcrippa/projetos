Materialize CSS é um framework de front-end baseado nos princípios de Material Design, desenvolvido pelo Google. Ele fornece uma coleção de estilos CSS, componentes e plugins JavaScript que facilitam a criação de interfaces de usuário modernas e responsivas
Grid System: Sistema de grade utilizado para criar layouts responsivos. Ele é baseado em 12 colunas e permite ajustar o layout de acordo com diferentes tamanhos de tela.
